iOS 常用Xcode插件

Posted on 2014-10-30 | In iOS |

今天有兴趣看看Xcode插件，以前都是不用插件的，觉得麻烦。虽然工具有点麻烦 有时候也会出错，但是使用好工具能够节约你不少时间

一、插件管理工具
安装Alcatraz,这是一个好东西,各种插件都可以在里面下载安装，非常好用.
在终端输入一下即可自动安装:

二、我比较喜欢的插件

1.BBUDebuggerTuckAway

在你编写代码的时候,它可以自动收缩底部的调试栏

2.CocoaPods

第三方库 管理工具CocoaPods的插件，以前都是输入命令行，现在可以用鼠标点点就可以了

3.KSImageNamed

在输入UIImage的时候会自动识别app下得图片资源文件，很方便

4.OMColorSense

输入颜色代码的时候可以预览颜色，同时也可以调用出颜色选择面板进行颜色选择

5.VVDocumenter
很多的注释插件，方便又好看
ps:Some users reported the setting panel can not be opened if installing from Alcatraz, so I suggest to build the plugin yourself until this issue get fixed
如果遇见用Alcatraz安装不能打开VVDocumenter设置，可以删除~/Library/Application Support/Developer/Shared/Xcode/Plug-ins目录下的VVDocumenter，然后到https://github.com/onevcat/VVDocumenter-Xcode下载 用Xcode build 安装一次，OK!

6.XCFixin_CustomizeWorningErrorHighlights
以前Xcode双击 都是下划线，这个插件可以高亮提示，看起来很方面，下滑虚线太难找了

7.SCXcodeMiniMap
一个Xcode插件，可以在当前的窗口内创建一个代码迷你地图，并在屏幕上高亮提示。

8.XAlign
一个用来对齐常规代码的Xcode插件，十分强大的自定义对齐模式。这里是一个对齐模式示例，模式文件在main/main/patterns.plist.

  1. Backlight
    某些插件从表面上看似乎无足轻重或者有些简陋，但只要大家加以合理运用、它们都将迸发出惊人的能量。Backlight就是这样一款插件。它惟一的能力就是对当前正在进行编辑的代码行加以高亮显示——没错，虽然看起来简单，但却极具价值

10.Injection for Xcode
一个神奇的Xcode 插件，能让应用在运行的时候做出的小的改变立马体现效果，而不需要重新编译。。。

11.RMConnecter
在上传AppStore时需要填写app的描述信息，此软件能很方便的填写这些信息。

12.XcodeBoost
一个辅助代码编辑插件。支持高亮选中、批量选中方法和方法名、根据选中的方法批量生成方法声明、高亮正则搜索等功能。

13.Code Pilot
一个能够快捷模糊查询代码，快速进入你想编辑的代码

14.XToDo
一个快捷记录你想将要做什么，以及解决了什么的标签，方便以后回看。
