病了……。

发表于 2008-07-31 | 分类于 Lifes |

昨晚就觉得头疼，感觉自己的老袋要炸了。我在给外婆说我头疼，外婆说吃头痛粉。我想到头痛粉，心里都不爽，想起我们高考时也吃头痛粉，心里有九千个不愿意。心想睡一觉明天就没事了。 晚上更加厉害了，我想这下可死了。终于慢长的一夜过了，早上怎麽也起不了床，老袋痛得要命。我就更外婆说那里有头痛粉，实在没办法，只有吃那东西了。。吃了后，感觉还不错，希望自己能快点好起来。

大学

发表于 2008-07-29 | 分类于 Lifes |

就要进入大学了，心里感觉轻松多了。但大学也不是想象中的好，人们都说大学好耍，有的却说不一定。大学要做的事很多，比如对我来说比较难的就是过英语四级。有位大学生给我说要想在大学里好耍，首先得把英语四级过了，就随便你怎么耍了。 大学，是我自由发挥的场所，我要让我的大学成为人生中最美好的一段时光。梦想在那里开花。

今天又地震了。。。

发表于 2008-07-24 | 分类于 Lifes |

今天上午 突然， 一个外地朋友说“我昨天晚上地震了你知道吗？我们的火车在路上停了一个多小时，我现在在江邮。” 我回答：“不知道，不会吧，在那里？” 到了中午，大概三点，在家看电视，看累了，想回屋休息，刚到就感觉在房子在摇晃，我又想起朋友说的话。才知道是地震。当时，把我吓坏了，我衣服还没穿。却还冷静的对弟弟说：“别怕，震不垮”。后来，我用手机查了下，是四川和陕西的交界处发生了5.6级的余年震。

一只小蜜蜂闯入了我的卧室

发表于 2008-07-23 | 分类于 Lifes |

一大早，本来挺安静的，没想到一只蜜蜂进了我的屋。进了没什么，可恶的是它一直在那里飞来飞去，一直“嗡嗡”的叫个不停。气得我吐血，干什么麻，“小东西，小心老子我劈了你，敢打扰我的美梦。” 气 气 气死我了，看我怎么收拾你！我坐了起来，可想要是没打到，它反过来咬我一口我可受不聊。哎，拿它没法，气死我了。 不行，没那么容易饶了你。奇怪，怎么没听见声音了。啊，啊，啊……好哇，逃跑了，看你识相。但心头的恶气没出，觉也睡不了了，只有无奈！

又是如此，没救啦！

发表于 2008-07-22 | 分类于 Lifes |

今晚下了场大雨，非常大。开始还是握着手机聊天，从八点一直聊到十一点，聊了很多。他们那里也在下雨，而且一样很大。告别晚安后，我还是没睡意。于是又拿起手机上网，去看点什么的。一直看，看到了一点钟，突然，一声大雷声惊动了我的心。 不可能又是一个不眠的夜吧，不会的。快睡吧，快睡吧，努力的说服自己！ 雨听了……，不知道，慢慢的，我已感觉自己神智不清了。好吧，再见，朋友们。明晚见咯。。
