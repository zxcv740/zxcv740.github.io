MacDrive 9

Posted on 2014-10-14 | In Hackintosh |

MacDrive是一款可以让PC机在Windows环境下读取Mac磁盘格式(HFS+分区)的软件,最新版本已经支持了Windows8操作系统!

通过它我们可以将MacOS与Windows文件共享的软件.支持WindowsME/98/95/NT/4.0/2000/XP与Mac系统文件交换,包括软盘、Zip、Jaz、CD-ROM、CD-R以及硬盘驱动器等.通过该软件从Windows直接访问Mac磁盘.打开、编辑、保存、复制文件，还支持在Windows下格式化Mac磁盘等.非常有有用的东西，用来在Win系统下研究Mac(比如精简原版),真是太方便了

2.下载原版镜像,鼠标右击显示包内容–>Contents–>SharedSupport

终端开启显示隐藏文件defaults write com.apple.finder AppleShowAllFiles YES

3.双击打开,右击打开BaseSystem.dmg

4.回到磁盘工具,操作如图,点击恢复

等待几分钟…

5.恢复完成后,打开恢复的OX X Base System 修改名字Mac 10.10和图标

到U盘那目录(即Mac 10.10)/System/Installation/下面有个Packages的替身（快捷方式），删除！

6.打开OS X Install ESD 找到Packages复制到刚刚删除的目录,同时拷贝BaseSystem.chunklist和BaseSystem.dmg到Mac 10.10下


7.加入extra文件夹,这样就制作完成了。结构图如下
